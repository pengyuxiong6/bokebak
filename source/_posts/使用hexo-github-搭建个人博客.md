## 安装步骤
1,全局安装hexo-cli
```
    npm install hexo-cli -g

```
2,初始化项目
```
    hexo init ./当前文件夹
```
3,hexo常用的几个命令：
```
    hexo n '文章的名称'      #创建文章
    hexo g                  #生成静态页面
    hexo s                  #启动本地服务,进行预览
```
hexo s 后直接打开浏览器输入地址进行预览即可。
